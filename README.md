MORSE CODE
It is a character encoding system used in telecommunication. • Morse code focuses on encoding systems, Basic elements, Transmission mediums, efficiency and speed. • Encoding systems can be represented with letters, numbers and symbols using short and long signals. • With Basic elements, dots(dits) can be represented in short signals, while dashes(dahs) is represented in long signals. • Transmission medium s can be transmitted through electric current, radio waves, visible light or sound waves. • Efficiency and speed have elements that are proportional to character frequency English. • Transmission rate(speed) is specified in groups per minute (words per minute)
